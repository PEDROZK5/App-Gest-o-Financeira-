
# App Gestão Financeira

Aplicativo desenvolvido em React Native para auxiliar no controle financeiro pessoal.

## Funcionalidades

- Login de usuários
- Cadastro de usuários
- Dashboard financeiro
- Controle de receitas e despesas
- Navegação entre telas
- Interface responsiva

## Tecnologias Utilizadas

- React Native
- JavaScript
- Expo
- React Navigation
- Node.js
- Git
- GitHub
